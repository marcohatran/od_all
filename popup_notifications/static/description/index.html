Popup Notifications

Use popup notification to develop your modules

The app is a tool of generating popups similar to calendar events alarims in your own modules:

  • Popup is generated Odoo for users
  • Popup is invoked each 3 minutes by standard Odoo javascript cron jobs
  • You may close the popup just by clicking "Ok"
  • Notifications do not depend on a definite object. You may create in any place disregarding Odoo models

Example of use

values = {'status': 'draft','title': u'Be notified about','message': "Do not forget to send notification in 21.04.17",'partner_ids': [(6, 0, [1, 2, 3])]} self.env['popup.notification'].create(values)
